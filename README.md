# AI Resume Customizer

AI-powered resume customization tool that tailors resumes according to a Job Description using Gemini AI.

## Features

- Upload Resume PDF
- Paste Job Description
- ATS-Friendly Resume Optimization
- Resume Analysis
- Download Customized Resume

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- PyMuPDF

### AI
- Gemini 2.5 Flash

## Run Locally

Backend:

```bash
uvicorn main:app --reload
```

Frontend:

```bash
npm run dev
```
